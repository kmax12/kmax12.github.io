How it Works

Using new technology, FeatureLab opens up the world of data science to anyone who has questions about data.

Transform data

The process of transforming raw data into useful representations is at the core of data science. Even with the most advanced technologies, human involvement is still needed to brainstorm, implement, and evaluate ideas for representing data to a machine learning algorithm.

Using a new technology called Deep Feature Synthesis, FeatureLab mines the relationships in your data to build these transformations automatically.

Build optimized models

FeatureLab's approach uses artifical intelligence to emulate human data scientists. This means we automatically choose the best variables to use, as well as build models using those variables.

After building an initial model, FeatureLab recommends which refinements are best suited for your dataset, allowing you, the data expert, to choose the best path forward.

Operationalize predictions

With a predictive model in hand, FeatureLab helps easily find insights and put your data to use. It is easy to simulate new inputs to test hypotheses or export batch predictions to use in another application.

"FeatureLab starts with a high-performance, auto-generated model and gives you an innovative interface to improve the solution."
