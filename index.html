How it Works

Using new technology, Workbench opens up the world of data science to everyone

The process of transforming raw data into useful representations is at the core of data science. Even with the most advanced technologies, human involvement is still needed to brainstorm, implement, and evaluate ideas for representing data to a machine learning algorithm.

Workbench's approach uses artifical intelligence to emulate human data scientists. This means automated discovery of the most valuable transformations, as well recommendations on how to improve established models.

Using a new technology called Deep Feature Synthesis (DFS), Workbench mines the relationships in your data to build these transformations automatically.

With an initial model built, Workbench recommends which refinements are best suited for your dataset, allowing you, the data expert, to choose the best path forward.
